# ALPHA Shop 購物車 Vue.js
透過 Vue.js 打造的三步驟表單

## Preview
![](https://i.ibb.co/jZGd4QZ/localhost-8080-2.png)

## Features
- 結帳區表單分成三步驟：寄送地址、運送方式和付款資訊
- 使用者用「上一步」、「下一步」按鈕來切換步驟
- 使用者可以選擇「運送方式」，運費能被反映到購物籃中
- 使用者在「購物籃」中可以調整商品數量，金額反映到小記中
